Playground 둘러보기

Playground 둘러보기

Role

  • System : 모델 사용 전 역할/설정 등 구성
  • User : 실제 메시지
  • Assistant : ChatGPT(User와의 대화 저장, 이전 대화 문맥)

설정

  • Temperature : 대답의 random 정도(높을수록 랜덤하게, 낮을수록 일관되게)
  • Top P : 샘플링시 확률이 상위 몇 %까지인 값만 취할까 (그냥 1로 놓고 쓰는 게..) 
  • Frequency penalty: 동일한 단어/문구를 반복하지 않도록
  • Presence penalty : 다양한 범위에 토큰을 포함시키도록 함

* OpenAI 문서에서는 temperature와 TopP를 동시에 변경하지 말라고 가이드함
